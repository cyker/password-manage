# password-manage
个人密码管理器

===

国内科技公司一贯的尿性和市面上密码管理器用的不太顺手，自己写了这款个人密码管理器。先说下优缺点。

缺点：

	1、不能自动获取浏览器上的链接、注册信息，全部得手动复制
	2、用户界面做的很差，功能都是自己修修补补，大多功能没实现，够用就好。bug自然也多
	3、数据库没有加密

优点：

	1、开源，无联网，放心
	2、使用 python 原生库，可不使用其他库
  
  
  
  详细说明及反馈请转[我的博客](http://blog.c1ker.top/2018/02/24/个人密码管理器/)
